### 🛡️ OpSec Toolchain

The repositories below form a lightweight, high‑assurance toolchain for secure file encryption, trusted viewing, and encrypted filesystem management – everything you need for operational security and cryptographic research, all written in C.

| Repository | Description | |
|:-----------|:------------|:--|
| **⚡ light** | Hybrid Post‑Quantum KEM • Streaming AEAD • Suite A‑Style Simulation. Shining a little light… with a touch of Obscurity. <br/><br/>`#PQC #AEAD #StreamingCipher #OpSec` | `C` |
| **📦 axis** | Axis-512 — Hybrid Post‑Quantum File Encryption. The unbreakable, streaming, post‑quantum‑ready file encryption tool. <br/><br/>`#PQC #Keccak #XChaCha20 #TMR` | `C` |
| **🔐 secure_mount** | A menu‑driven, secure wrapper for gocryptfs that simplifies initializing, mounting, and unmounting encrypted filesystems with zero‑effort hardening. <br/><br/>`#gocryptfs #FUSE #FilesystemEncryption` | `C` |
| **🖼️ viewer** | RAM‑only image viewer for classified/sensitive JPEG and PNG files. Loads images directly into non‑swappable (`mlock`) memory, securely wipes all pixel data on exit, and never writes decrypted data to disk. <br/><br/>`#MemorySecurity #X11 #JPEG #PNG` | `C` |
| **🌀 obscurity** | A hardened, research‑grade implementation of a novel sponge‑based cryptographic primitive designed to simulate the engineering discipline, operational security patterns, and threat‑model assumptions of classified cryptographic systems (e.g., NSA Suite A‑style ciphers). <br/><br/>`#SpongeCipher #ARX #TMR #ConstantTime` | `C` |
